How to Submit a Dataset

If you have a dataset you are interested in seeing on this website please choose an option below to proceed.

Option 1 (preferred): Using Github Repositories

Please use this option if you are familiar with setting up a Github repository and if your dataset does not contain any single files that are larger than 4GB.

  1. Fork the dataset template repository here
  2. Populate the template repository with your data
  3. Create a pull request on the PACT Github. Your pull request should include the following changes to the PACT Repository:
    • Your dataset card added to the datasets.html file (with correct 'Download' and 'View on Github' links)
    • Your dataset's repository added as a submodule under the top-level datasets folder
  4. Add the following tag to your pull request: Task: Dataset Submission
  5. A member of the PACT Datasets and Evaluation team will review your pull request, validate your dataset and notify you if it has been added or if changes are required.

Option 2: Without Github Repositories

If you do not wish to use Github or your dataset is too large for Github's large file share (LFS) feature, please email the PACT Datasets and Evaluation team at pactdatasets@gmail.com with the dataset name and a brief description of how it was collected. Please either attach your dataset directly to the email or else provide a link to where it can be accessed by the PACT Team.
